# dasWetter
An app built in React to forecast the weather using [OpenWeather](http://api.openweathermap.org)


### Installation

First install all dependencies for the project using:
```
$ yarn install
```

then, run the server with:

```
$ yarn start
```


### Technologies
* React 17.0.2
* React-dom 17.0.2
* Redux 7.2.6
* Plotly.js 2.9.0
* Parcel 2.2.1
